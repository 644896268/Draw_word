# Draw_word
A small script written with jupyter that converts pictures into characters and saves them in TXT.

一个使用jupyter写的小脚本，可以把图片转换为字符并保存在TXT中


##过程：

1导入图片 2调整大小 3转为灰度图 4使用字符替代灰度像素
